SWITCH STATEMENTS

A switch statement is unique from every previous statement we've learned so far, with a completely new syntax: 

		
			switch (expression){
				case label: 
					statements; 
					break;
				case label: 
					statements; 
					break; 
				...
				default:
					statements; 
					break;
				}

The switch statement begins with the keyword "switch." It functions by comparing the value of an expression to a value contained within a special statement called a case label. A case label in a switch statement represents a specific value; in our syntax example, you see that each case label is followed by one or more statements that are executed if the value of the case label matches the value of the switch statement's expression. The break statement is important in a switch statement, because unlike an if, if else, or else if statement, a switch statement will run through every case label by default unless the break statement is present to end the statment. A switch statement is great for situations when you're working with many different conditions and outcomes. Let's look at an example: 

		
			var day;
			switch (new Date().getDay()) {
				case 0:
				    day = "Sunday";
				    break;
	  			case 1:
				    day = "Monday";
				    break;
				case 2:
				    day = "Tuesday";
				    break;
				case 3:
				    day = "Wednesday";
				    break;
				case 4:
				    day = "Thursday";
				    break;
				case 5:
				    day = "Friday";
				    break;
				case 6:
				    day = "Saturday";
				}

			document.getElementById("message").innerHTML = "It's" + day ;

This is a relatively simple example of a switch statement which displays the day of the week. First, the variable day is declared. Then, we enter our switch statement. The conditional statement uses the getDay() method, which uses the javascript date object to return the day of the week as a number 0-6. We then begin working through our case labels. The first case evaluates the expression with the label "0" which equates to Sunday. If the values match, the statement within the case 0 block is run, meaning the variable day is assigned the string value "Sunday," and the program breaks from the statement. If the values do not match, the code will continue to check each case label with the statement until it either finds a value or the statement ends. Some switch statements include default statements, as a premeditative attempt to mitigate runtime errors, bugs, and other code issues.
