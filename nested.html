NESTED CONDITIONAL STATEMENTS

If you've ever studied logic or computer science, you may be familiar with nested statements. If not, you will be shortly. This tutorial specifically will focus on nested if and if/else statements, which are made simply by putting an if statement or if/else statement inside of another if or if/else statement to test for further conditions and create more specific conditional results. Putting an if statement inside an if or if/else statement is called a nested if statement, while putting a if/else statement inside an if or if/else statement is called a nested if/else statement. Let's look at the syntax for a nested statement:

		
			if (expression) {
				statements; 
			}
			else {
				if (expression) {
					statements; 
				}
				else {
					statements;
				}
			}

The syntax for this relatively simple nested statement is exactly the same as our previous examples; the only difference is we now have the oppurtunity for 3 different outcomes instead of just two. Let's revisit our example: 

		
			var hour = new Date().getHours(); 
  			var message;
  			if (hour < 12) {
  				message = "Good morning!";
  			}
  			else {
  				if (hour < 16) {
					message = "Good afternoon!";
  				} 
  				else {
  					message = "Good evening!";
  				}
  				
  			}

This example is very similar to our previous statement; however we've added a conditional block to further elaborate the message given for the second half of the day. Within our original if statement, we've added a nested if/else statement. This program first checks to see if the hour is less than 12; if that returns true, the message variable is assigned "Good morning!" If it returns false, our program will then move to the nested if statement and check to see if the hour is less than 16 (earlier than 4 p.m.) If this returns true, message will be assigned "Good afternoon!" If it returns false, this means the hour is greater than 16, or it is later than 4, and the program will run the command block of the nested else statement, which assigns message the string: "Good evening!" Again, this is not a very complex example, but nested statements open up a world of oppurtunity when it comes to decision making structures! Click "Next" to continue!
