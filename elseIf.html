ELSE IF STATEMENTS

An else if statement is a compact version of a nested if/else statement. It combines an else statement with it's nested if statement which requires fewer characters and lines of code, and is easier to read. The syntax should look familiar by now: 

		

			else if (expression) {
				statements;
			}

This is really the exact same as saying: 

		

			else {
				if (expression) {
					statements;
				}	
			}

however, our else if construction takes up 3 lines, while the nested if statement takes up 5. For our else if example, let's look at one of the most common and useful applications of this topic: creating backwards compatible event listeners. 

		
			var submitButton = document.getElementById("button");
			//if addeventlistener method exists for the button, the method is executed 
			if (submitButton.addEventListener) {  
				submitButton.addEventListener("click", submitForm,  
					false);
			}
			//if addeventlistener does not exist it checks for attachevent method. 
			//if that exists it runs, if not it does nothing. 
			else if (submitButton.attachEvent) {
				submitButton.attachEvent("onclick", submitForm);
			}

This example begins by creating a variable named submitButton and using the getElementById() method to assign it the value of the element in the document with the ID "button." Then we go into our if statement, which checks to see if the addEventListener method exists for the button, and if that condition evaluates as true, or if the method does exist, the command block will fun using the addEventListener method. If the method does not exist, or the condition evaluates as false, the program moves on to the else if statement, which then checks to see if the attachEvent method is present for the button. If that conditional statement evaluates as true, the command block will run using the attachEvent method. If it does not exist, the program will return nothing.
