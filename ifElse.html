IF/ELSE STATEMENTS

Following the "If Statement", naturally, is the "If/Else Statement." The syntax for this is quite similar to an if statement with one additional keyword to define: 

		
			if (expression) {
				statements; 
			}
			else {
				statements;
			}

An if/else statement begins with the previous "if statement" syntax we saw on the last page; however, rather than returning nothing if the conditional statement evaluates to a falsy value, the if/else statement includes a second command block which will run in this scenario. This is the "else statement" and begins, obviously, with the keyword "else." Lets revisit our last example: 

		
			var hour = new Date().getHours(); 
  			var message;
  			if (hour < 12) {
  				message = "Good morning!";
  			}
  			else {
  				message = "Good afternoon!";
  			}

This example is essentially the same as the one on the last page with a very minor ammendment. After running through the first "if statement", we have an "else statement" with a different command block which defines the variable message as the string "Good afternoon!" In the event that the conditional statement evaluates as falsy, or, plainly, in the case that it is later than 12, the message which, again, is displayed somewhere else within the document, will display the words "Good afternoon!"
