IF STATEMENTS

One of the simplest and most common JS conditional statements is the "If Statement". The syntax is rather simple: 

		
			if (expression) {
				statements; 
			}

The first important part of an if statement is the keyword "if" which must begin the statement. The "expression" within this syntax example represents the conditional statement which is evaluated for a truthy or falsy value. The "statements" represents the code to be executed if the conditional statement returns a truthy value. If there is more than one line of code to be run we call this a "command block." Lets look at an example: 

		
			var hour = new Date().getHours(); 
  			var message;
  			if (hour < 12) {
  				message = "Good morning!";
  			}

This if statement first creates a variable with the name "hour" and uses the getHours() method to obtain the time of day and assign it as the definition of the variable. Next, a new variable named message is declared but is not defined. The definition of this variable will depend upon the results of the conditional statement.

The if statement begins with the keyword "if" followed by the conditional statement: "(hour < 12)"; this is the conditional statement that is evaluated. If this statement evaluates as true, or more plainly, if it is earlier than noon, the variable message will be assigned the value of the string "good morning!", which is likely to be displayed somewhere else in the document. This is likely one of the most simple forms of conditional statements, but it sets the stage for us to continue learning some more complex conepts! Click "Next" to continue!
